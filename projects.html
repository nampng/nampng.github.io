Senior Design Project - Helmet HUD System - 2nd place

My team developed a multifunctioning system that was capable of providing location information, teammate tracking, and wireless communications. All this information was then projected onto an augmented reality display, which allowed the user to still be situationally aware of the environment around them.

The heart of the system was a Raspberry Pi 4 with a BerryIMU v4 and LoRA transceiver. The Pi was responsible for wireless communications, calculations, and location services.

The augmented reality display was achieved by using a NorthStar Deck X headset, Intel RealSense T261 tracking module, and the Panda3D game engine.

There were a lot of pitfalls and obstacles when doing this project. I had to quickly learn on the spot and be proactive in protyping and testing as the project had time and budget constraints. We did not have the same amount of guidance that the corporate sponsored teams had, so we were usually poking around in the dark until we found something that worked. There were times where we doubted if we could even see our project to the end, but through determination we managed to create a working prototype and place among the top three at the Senior Design Expo.

This I've learned from this project:

  • Even More Python
  • Bluetooth (Was our original wireless communication method)
  • GPS
  • LoRA
  • Panda3D
  • Intel RealSense API
  • Hard work and determination is the key
